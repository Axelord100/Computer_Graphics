Dada una obra abstracta, se realizó esta tarea con el uso de OpenGL con Shaders y Textures bajo GLFW. 
Las siguientes características fueron utilizados para la realización:
	- Vertex shaders
	- Fragment shaders
	- Textures
	- Diferentes formas de armado de polígonos (TRIANGLES, TRIANGLE_FAN, POLYGON)

El código main se encuentra dentro de la carpeta "Tarea2". En esta carpeta se encuentra tanto el código
fuente en C++ "Tarea2.cpp" y se adjunta la solución de Visual Studio "Tarea2.sln". 
Del mismo modo se incluyen los FS Files y VS Files para poder utilizar la imágen "portal.jpg" como textura.
Para poder ver la textura en el programa se debe de poner en una carpeta con un archivo "atb_image.h". 
En este caso, se utiliza en un PATH local por lo que también debe de ser editado en la línea 1102. 

"arte_Abstracto.png" tiene la obra abstracta pedida a realizar. 

"Resultado.png" tiene el resultado final de la tarea. La decisión de los colores recae en la decisón de 
la imagen. Se buscaba tener tanto un contraste como una armonía entre cada figura/polígono realizad@. 
Se presenta una obra con aldededores de colores cálidos y un centro de colores fríos. Esta combinación
ayuda a contrastar el centro con los alrededores y genera una gran armonía con la textura utilizada.
La idea es tener una percepción de un portal.  

En "Ejemplo_Calculos.png" se tiene un ejemplo de cómo se realizaban los cálculos para así obtener una
simetría similar a la de la obra abstracta dada por el profesor. 

"Tarea2.txt" contiene una copia del código en formato .txt para poder copiar y pegar el código en
cualquier caso de funcionamiento erróneo o no se pueda visualizar el código correctamente. 

"Tarea2_2.docx" es una copia de los requerimientos y lineamientos para realizar esta tarea.
